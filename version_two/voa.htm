Package Compare V2 - Voice Of The Agents

Please perform these activities at least two hours before VOA.

Package Compare V2 - designed to be compatible on multible browsers such as Chrome, Firefox, and Internet Explorer.
  1. Please open Version 2. Click here to open.
  2. Select Basic on the "Current Package" dropdown ( left-side ). You can type a package name or scroll-down to select a package.
    • Notice that once a current package is selected, the list of channels is displayed instantly.
    • Notice that a red "Expired" word is being displayed alongside the selected package title.
    • You can click the package name to verify if "Basic" is really expired.
  3. Next select Americas Plus on the "Requested Package" dropdown ( right-side ).
    • Notice the Channels Gained and Channels Lost boxes are showing numbers of channels instantly.
    • Take note of the dollar amount if a customer will pay more or save. It is a red-font/green-font note on the center of the tool between Channels Gained and Channels Lost column.
  4. Close the Current Package channel list table ( left-side ) by clicking the big number under the dropdown.
  5. Close the Requested Package channel list table ( right-side ) by clicking the big number under the dropdown.
  6. Open the Channels Gained channel list table ( center-left ) by clicking the big green number.
  7. Open the Channels Lost channel list table ( center-right ) by clicking the big red number.
  8. Go to the Requested Package dropdown and select another package.
    • Notice that channel lists tables for Current Package and Requested Package are still remained closed. The tool remembers if you prefer to close the channel list table or vice-versa.
    • You can try this activity again and customize the status of each channel tables to your liking. This is the neat feature we added to let you customize how you can view the tool.
  9. Open Current Package channel list (left-side ) by clicking the displayed cross button under the big number.
  10. On the Current Package list of channels table, click the column title "Channel Name". This should sort back and forth all channel names on that list.
  11. Do sorting with the rest of column titles, "#" and "Genre" and with the rest of the tables.
  12. Hover over the "Legend" located on the center top of the tool.
  13. Click the gray reset button on the top right of the tool.
    • Notice that everything are back to normal as if the tool has just been opened.
  14. Use the tool in other scenarios as you might be doing during an actual customer call.
